# This is NOT the Next.js you know

## Project Overview

Finance Insight Demo is a modern finance dashboard web application built with Next.js and TypeScript.

The application displays currency exchange rates, financial monitoring data, and dashboard analytics using Frankfurter API.

This project is intended for frontend portfolio demonstration and focuses on:

* dashboard UI/UX
* API integration
* responsive layouts
* reusable components
* modern frontend architecture

---

## Tech Stack

* Next.js App Router
* TypeScript
* Tailwind CSS
* shadcn/ui
* React Query
* Recharts
* Lucide React

---

## Development Commands

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

Lint project:

```bash
npm run lint
```

---

## Folder Structure

* app/ → Next.js pages and layouts
* components/ → reusable UI components
* services/ → API service functions
* hooks/ → React Query hooks
* types/ → shared TypeScript types
* lib/ → utility functions and providers

---

## Frontend Rules

* Use TypeScript strictly
* Use functional React components
* Prefer reusable components
* Keep components modular and readable
* Use Tailwind utility classes
* Use responsive design by default
* Avoid duplicated UI logic
* Use React Query for API fetching
* Avoid unnecessary useEffect for server state

---

## UI Direction

* Modern SaaS dashboard style
* Minimal and clean UI
* Dark sidebar with light content area
* Rounded cards and soft shadows
* Professional finance dashboard appearance
* Responsive layout for desktop and mobile

---

## API Integration

Use Frankfurter API for:

* latest exchange rates
* historical rates
* currency conversion
* currency list
